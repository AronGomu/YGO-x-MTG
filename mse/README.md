# MSE data

English folder-form saves under `MSE_projects/*.mse-set/` are card source of truth.

Legacy monolithic aggregate was retired because it duplicated card data and could overwrite newer project edits. Frozen French snapshot remains at `mse/French/set` for reference only.

Use project workflows to edit cards:

- `add-ygo-card` for new conversion from official Yu-Gi-Oh! card;
- `update-card-from-ai` for supplied card definitions;
- `fix-mse-cards` after direct MSE edits;
- `normalize-card-formatting` for formatting-only updates.
