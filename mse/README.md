# MSE exports

Generated Magic Set Editor save files for the Yu-Gi-Oh. × Magic Cube.

- `yugioh_magic_yu_oh_cube_m15.mse-set` — primary export using the Magic M15 template.
- `yugioh_magic_yu_oh_cube_magic_new_fallback.mse-set` — fallback export using the older default Magic template if M15 is missing from your MSE install.
- `yugioh_magic_yu_oh_cube.mse-set` — original alias of the M15 export.

The set contains 77 cards parsed from Burning Abyss, Shaddoll, Necroz, utility cards, and Spellbook project documents.
